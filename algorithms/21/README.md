# Merge Two Sorted Lists

ID: `21`  
Name: `merge-two-sorted-lists` (`合并两个有序链表`)  
Url: [remove-nth-node-from-end-of-list](https://leetcode-cn.com/problems/merge-two-sorted-lists/)  
Pass: `2019-12-04`

## Description

将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

示例：

> 输入：1->2->4, 1->3->4  
> 输出：1->1->2->3->4->4
